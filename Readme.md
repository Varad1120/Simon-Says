# Simon Says
  
- this is the recreation of the very popular 'simon-says' game, made using html, css and javascript.  
## How to play  
- there are 4 boxes of different colors and one of them will flash, user will have to press on the flashed box then another box will flash, then user will have to press on the last flashed box by pressing through the sequence of all the flashed boxes, game will reset if the user presses a wrong box.  

## have fun playing
